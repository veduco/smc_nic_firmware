# Supermicro X10SRM-TF X550 Firmwares
The Supermicro X10SRM-TF is a micro ATX motherboard which supports Intel's E5 v3/v4 processors.

**Note:**  
Previous v3.50 in this repository was sourced from SMC.  
Current v3.70 in this repository was sourced from INTC with SMC's modifications applied during flashing via the EEPROM map file [x550.txt](x550.txt).
